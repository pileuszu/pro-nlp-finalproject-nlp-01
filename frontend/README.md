# 모두취업 프론트엔드 (Frontend)
 
모두취업 서비스의 사용자 인터페이스를 담당하는 프론트엔드 프로젝트입니다. Next.js 14와 AI 기술을 결합하여 구직자들에게 최고의 포트폴리오 관리 및 자소서 작성 경험을 제공합니다.

## ✨ 주요 기능

-   **AI 포트폴리오 분석**: GitHub 리포지토리, Notion 워크스페이스, PDF 파일을 분석하여 핵심 역량을 자동으로 추출합니다.
-   **스마트 채용공고**: 실시간 채용 공정 데이터를 시각화하여 제공하며, 필터링 및 페이징 기능을 지원합니다.
-   **AI 자소서 에디터**: 분석된 포트폴리오 데이터를 바탕으로 자기소개서를 작성하고 AI 첨삭을 받을 수 있습니다.
-   **프리미엄 UI/UX**: Framer Motion을 활용한 부드러운 애니메이션과 shadcn/ui 기반의 깔끔한 현대적 디자인을 제공합니다.

## 🛠 기술 스택

-   **Framework**: Next.js 14 (App Router)
-   **Language**: TypeScript
-   **Styling**: Tailwind CSS, shadcn/ui
-   **Animation**: Framer Motion
-   **State Management**: TanStack Query, Zustand
-   **Mocking**: MSW (Mock Service Worker)

## 🚀 시작하기

### 설치 및 실행

```bash
# 의존성 설치
npm install

# 개발 서버 실행
npm run dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000)을 열어 확인하세요.
