## 1. 프로젝트 개요
수많은 채용 공고 속에서 나에게 맞는 기업을 찾는 피로감과 반복되는 자기소개서 작성의 비효율을 해결하기 위한 AI 기반 취업 자동화 솔루션입니다. 단순히 공고를 나열하거나 문장을 교정하는 기존 플랫폼의 한계를 넘어, 사용자의 포트폴리오와 GitHub 레포지토리를 심층 분석하고 실시간 채용 데이터(RAG)를 결합하여 최적의 기업 추천과 맞춤형 자소서 초안을 동시에 제공합니다.

## 5. LLM 파이프라인

### 5-1. 포트폴리오 파이프라인
다양한 소스(Notion, GitHub, PDF, Blog)에서 텍스트를 추출하고 정제합니다.
- **텍스트 추출**: Surya OCR을 통해 100% 정확도로 이미지 내 텍스트 확보.
- **텍스트 전처리**: Null Byte 필터링, 800자 단위 Chunking (160자 중첩).
- **구조화**: HyperCLOVA X (HCX-007)의 Structured Output 기능을 활용하여 고정된 JSON 스키마로 데이터 변환.

### 5-2. 공고 파이프라인
방대한 채용 공고를 수집하고 하이브리드 검색을 통해 추천합니다.
- **데이터 수집**: 사람인, 자소설닷컴, 인디스워크 등 주요 플랫폼 네트워크 트래픽 분석 및 구조적 크롤링.
- **추천 로직**: 
  - **Hybrid Search**: Dense(BGE-M3) + Sparse(BM25 + Kiwi) 결합.
  - **Multi-Query**: 기술 스택, 문제 해결, 커리어 맥락 관점의 쿼리 생성.
  - **Reranking**: HyperCLOVA X Reranker API를 통한 증거 기반 매칭.

### 5-3. 자소서 파이프라인
포트폴리오 기반 경험을 기업 요건과 연결하여 고품질 자소서를 생성합니다.
- **전략 설계**: STAR 구조 모범 예시(Few-shot) 적용 및 Temperature 0.0 설정으로 환각 방지.
- **모드 지원**: 아웃라인 전용 모드(--outline) 및 소제목 포함 생성 모드(--subheading).
- **검증**: Pydantic 스키마를 통한 구조화된 출력 제어 및 재현성 95% 확보.

## 6. 자체 평가 의견
- **잘한 점**: 최소 기능(MVP) 조기 구현 및 프롬프트 엔지니어링의 효과 확인.
- **시도했으나 아쉬운 점**: 인성적 경험(협업 등) 추출의 한계, Fine-tuning 미진행.
- **시사점**: 전체 서비스 흐름을 고려한 통합 설계의 중요성 및 사용자 중심 설계 이해.
